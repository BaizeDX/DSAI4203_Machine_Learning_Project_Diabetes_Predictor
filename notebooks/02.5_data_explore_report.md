### Data Exploration Summary (Notebook 01)

Before building any predictive models, we conducted a concise exploratory data analysis to understand the dataset's structure and identify the main modeling challenges.

The dataset contains 700,000 training samples and 300,000 test samples. The training file has 26 columns, including both numerical and categorical variables plus the target column. The target variable `diagnosed_diabetes` is imbalanced: approximately 62.3% of samples are labeled as diabetic (class 1), while 37.7% are non-diabetic (class 0). This class ratio of about 1.65 means the positive class is the majority, which affects how model performance should be interpreted.

Analysis of numerical features showed that `family_history_diabetes` has the strongest positive correlation with the target (0.211), followed by `age` (0.161) and `systolic_bp` (0.107). These variables emerged as the most obviously predictive signals at the exploration stage. Outlier detection also showed that some features contain extreme values, particularly `physical_activity_minutes_per_week` with 4.78% outliers, suggesting that later modeling steps should be robust to distributional irregularities.

The categorical feature set includes gender, ethnicity, education level, income level, smoking status, and employment status. Initial examination suggested that these distributions are reasonably balanced, with no single category dominating so strongly that simple frequency effects alone would explain the target.

The exploration also confirmed that there are no missing values in the dataset, which simplifies the preprocessing pipeline. Overall, the feature distributions appear broadly plausible and consistent with the medical context of the problem.

### Baseline Model Results (Notebook 02)

A decision tree classifier was established as the first baseline model using default-style settings with a maximum depth of 5. The model was evaluated on a 20% holdout validation split and achieved an AUC of 0.6825.

The classification report revealed a clear imbalance-related weakness. The model achieved a recall of 0.88 for the diabetic class but only 0.30 for the non-diabetic class, indicating a strong bias toward predicting the majority class. Precision was 0.68 for diabetic cases and 0.60 for non-diabetic cases.

Feature-importance analysis from the decision tree showed that only three features contributed meaningfully to predictions: `family_history_diabetes` (44.8%), `physical_activity_minutes_per_week` (29.9%), and `age` (21.8%). The remaining features had negligible importance, suggesting that the shallow tree structure was not rich enough to capture more complex relationships in the data.

These baseline results established a useful reference point for later model development and highlighted two immediate priorities: handling the class imbalance more carefully and moving toward models that could use a larger share of the feature set more effectively.
