### Summary of Experimental Improvements

Experiment 11 introduced several methodological improvements that addressed the limitations identified in earlier experiments. Through 5-fold cross-validation and Out-of-Fold evaluation, we obtained more reliable model performance estimates. The AUC scores across the five folds were 0.72688, 0.72532, 0.72614, 0.72766, and 0.72652, yielding a mean of 0.72650 with a standard deviation of only 0.00078. This high consistency indicates strong generalization ability. The final Out-of-Fold AUC of 0.7265 is marginally better than the best earlier result (0.7264), but more importantly, the evaluation method itself is substantially more reliable.

### Comparison of Early and Improved Experiments

Validation Method
- Early: Fixed 20% holdout validation set, reused across multiple experiments
- Improved: 5-fold stratified cross-validation, each sample validated exactly once

Evaluation Metric
- Early: Single validation set AUC (potentially optimistic)
- Improved: Average of five fold AUCs and Out-of-Fold AUC (more stable and realistic)

Feature Processing
- Early: Created interaction features and removed original features, causing information loss
- Improved: Retains all original features while adding interaction features as complements

Categorical Encoding
- Early: Manual One-Hot encoding with `pd.get_dummies()`, risk of train/test column mismatch
- Improved: XGBoost native categorical support, single column per feature, automatic alignment

Overfitting Control
- Early: Manual selection of tree count
- Improved: Early stopping with patience of 100 rounds, automatic optimal tree determination

Result Reliability
- Early: Potentially optimistic due to repeated use of same validation set
- Improved: More reliable estimates through cross-validation and consistent performance across folds
